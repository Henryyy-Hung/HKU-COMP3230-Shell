# HKU-COMP3230-Shell

## Introduction
  * Simulate a shell base on linux terminal by fork(), pipe(), signal handling and so on.
  * Score: 11.5/12 (compulsory), 4/4 (bonus). 

## Quick Start

  1. git clone https://github.com/Henryyy-Hung/HKU-COMP3230-Shell
  2. cd ./HKU-COMP3230-Shell/src
  3. make all
  4. ./3230shell
